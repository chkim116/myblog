## myblog

리액트와 node js를 활용한 블로그

# 목표

- About me, 스택 설명 화면 구현
- S3 연동 / 이미지 업로드(quill)
- 회원가입 유효성 검사 (비밀번호, 아이디 글자 수)
- https://www.youtube.com/watch?v=IYCa1F-OWmk (페이징네이션 구현)
